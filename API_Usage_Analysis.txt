STUDENT ROLE
===================================================================================
API Endpoints được sử dụng:

• GET /api/student-profile/{userId} - Lấy hồ sơ học viên
• POST /api/student-profile/create - Tạo hồ sơ học viên  
• PUT /api/student-profile/update/{userId} - Cập nhật hồ sơ

• GET /api/course - Lấy danh sách khóa học (có phân trang, filter)
• GET /api/course/{courseId} - Chi tiết khóa học
• GET /api/course/{courseId}/instructors - Lấy giảng viên của khóa học

• GET /api/enrollments/my-enrollments - Khóa học đã đăng ký
• POST /api/enrollments/enroll - Đăng ký khóa học
• DELETE /api/enrollments/unenroll/{courseId} - Hủy đăng ký

• GET /api/lessons/by-course/{courseId} - Bài học theo khóa học
• GET /api/documents/lesson/{lessonId} - Tài liệu theo bài học

• GET /api/tests/by-lesson/{lessonId} - Bài test theo bài học
• GET /api/tests/user/{userId} - Test của người dùng
• GET /api/tests/{testId} - Chi tiết bài test
• POST /api/tests/auto-create?userId={userId} - Tạo bài test tự động

• GET /api/test-attempts/by-user/{userId} - Lịch sử làm bài
• GET /api/test-attempts/{attemptId}/with-score-summary - Kết quả làm bài chi tiết
• POST /api/test-attempts/start - Bắt đầu làm bài
• POST /api/test-attempts/submit - Nộp bài

• GET /api/test-template-types - Loại template bài test
• GET /api/test-templates/by-type/{testTemplateTypeId} - Template theo loại
• GET /api/test-template-configs/by-template/{templateId} - Cấu hình template

• GET /api/lesson-progress/completion-rate - Tỷ lệ hoàn thành khóa học
• GET /api/lesson-progress/by-user-lesson - Tiến độ theo bài học
• POST /api/lesson-progress - Tạo/cập nhật tiến độ

• GET /api/livestreams - Danh sách livestream
• GET /api/livestreams/{id} - Chi tiết livestream
• GET /api/livestreams/{id}/can-join - Kiểm tra quyền tham gia
• GET /api/livestreams/{id}/join-token - Lấy token tham gia

• GET /api/study-plans/get-by-studentid/{studentId} - Kế hoạch học tập
• GET /api/study-plan-items/get-by-plan/{planId} - Chi tiết kế hoạch

• GET /api/payment/history/{userId} - Lịch sử thanh toán
• GET /api/payment/credit-history/{userId} - Lịch sử tín dụng
• POST /api/payment/create-credit-purchase - Tạo đơn mua tín dụng
• GET /api/payment/check-credit/{userId}/{amount} - Kiểm tra tín dụng

• GET /api/conversations/my-conversations - Cuộc hội thoại của tôi
• POST /api/conversations/create - Tạo cuộc hội thoại
• POST /api/conversations/send-messages/{conversationId} - Gửi tin nhắn

• PUT /api/users/{userId} - Cập nhật thông tin người dùng
• POST /api/users/{userId}/avatar - Cập nhật avatar

INSTRUCTOR ROLE

API Endpoints được sử dụng:

• GET /api/lessons/by-course/{courseId} - Bài học trong khóa học
• POST /api/lessons/{courseId} - Tạo bài học mới

• GET /api/tests/by-lesson/{lessonId} - Test theo bài học
• PATCH /api/tests/{testId}/status - Cập nhật trạng thái test
• POST /api/tests/by-lesson/{lessonId} - Tạo test cho bài học

• GET /api/livestreams - Livestream của giảng viên
• POST /api/livestreams - Tạo livestream mới
• PUT /api/livestreams/{id} - Cập nhật livestream
• DELETE /api/livestreams/{id} - Xóa livestream

• GET /api/instructor-profile/{userId} - Hồ sơ giảng viên
• PUT /api/instructor-profile/update/{userId} - Cập nhật hồ sơ
